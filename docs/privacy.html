Privacy Policy

Privacy Policy

Effective Date: November 25, 2025

Your privacy is absolute. The CSS Selector Specificity Calculator runs entirely in your browser. We do not collect, store, transmit, or log any data — period.

1. No Data Collection

This tool processes all CSS selectors locally on your device using pure JavaScript. We do not collect:

  • Any CSS selectors you enter
  • IP addresses, browser fingerprints, or device information
  • Usage statistics or analytics
  • Cookies, localStorage (beyond temporary session), or any tracking pixels

2. 100% Client-Side Operation

Every calculation happens in your browser. When you:

  • Type a selector → processed instantly in memory
  • Click “Copy Results” → uses the browser’s native clipboard API
  • Share a URL → only encodes selectors in the URL fragment (#?q=…)

Nothing is ever sent to any server.

3. Third-Party Services

We use only Bootstrap 5 and Bootstrap Icons via official CDN (jsDelivr) for styling. These CDNs do not track individual users on static GitHub Pages sites. No analytics, no ads, no telemetry.

4. Data Security

Because no data leaves your device, there is no server to hack, no database to breach, and no logs to leak. Your selectors remain private by design.

5. Children’s Privacy

This tool is safe and appropriate for users of all ages. No personal information is ever requested or stored.

6. Changes to This Policy

As a static open-source project, any change to privacy practices would require a new version. Updates will be clearly announced on GitHub and reflected here.

7. Contact

This tool is open source and community-driven. Questions? Open an issue on our GitHub repository.

We believe privacy isn’t a feature — it’s the default.
Use this tool with complete confidence. Your CSS stays yours.
